# Wrikka.com

This is the source code for my personal portfolio website, [wrikka.com](https://wrikka.com), built with Nuxt 3. It showcases my projects, blog posts, and professional profile.

## âœ¨ Features

- **Nuxt 3**: Built with the latest version of the progressive Vue.js framework.
- **UnoCSS**: A utility-first CSS framework for rapid UI development.
- **Content-driven**: Blog posts are written in Markdown.
- **GitHub Integration**: Projects are dynamically fetched from my GitHub profile.
- **Dark Mode**: Supports both light and dark themes.

## ðŸš€ Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/newkub/wrikka-com.git
   cd wrikka-com
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Create a `.env` file and add your GitHub username and a personal access token:
   ```env
   GITHUB_USERNAME=your-username
   GITHUB_TOKEN=your-personal-access-token
   ```

4. Start the development server:
   ```bash
   bun run dev
   ```

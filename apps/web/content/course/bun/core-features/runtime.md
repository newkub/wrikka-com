# Runtime

Bun's JavaScript runtime with built-in TypeScript support.

## Key Features

- **TypeScript Support**: Run TypeScript files directly without compilation
- **Web APIs**: Built-in support for fetch, WebSocket, and more
- **Node.js Compatibility**: Works with most Node.js packages
- **Fast Startup**: Starts up 4x faster than Node.js

## Run TypeScript Files

```bash
bun run index.ts
```

## Automatic Transpilation

Bun automatically transpiles TypeScript on the fly, so you don't need a build step during development.

## Module Support

You can also use ES modules and CommonJS modules seamlessly.

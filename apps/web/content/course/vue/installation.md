# Installation

Install Vue in your project.

## npm

```bash
npm create vue@latest
```

## yarn

```bash
yarn create vue
```

## pnpm

```bash
pnpm create vue
```

## bun

```bash
bunx create-vue
```

## CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

# Installation

Install React in your project.

## npm

```bash
npx create-react-app my-app
```

## Vite

```bash
npm create vite@latest my-app -- --template react
```

## TypeScript

```bash
npx create-react-app my-app --template typescript
```

## Next.js

```bash
npx create-next-app@latest
```

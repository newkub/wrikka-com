# Utility Classes

Use utility classes for styling.

## Spacing

```html
<!-- Margin -->
<div m-4>Margin all sides</div>
<div mt-4>Margin top</div>
<div mb-4>Margin bottom</div>

<!-- Padding -->
<div p-4>Padding all sides</div>
<div pt-4>Padding top</div>
<div pb-4>Padding bottom</div>
```

## Layout

```html
<!-- Flexbox -->
<div flex items-center justify-center>Center content</div>
<div flex gap-4>Flex with gap</div>

<!-- Grid -->
<div grid grid-cols-4>4 columns</div>
<div grid grid-cols-2 md:grid-cols-4>Responsive grid</div>
```

## Typography

```html
<!-- Text -->
<div text-xl>Extra large text</div>
<div font-bold>Bold text</div>
<div text-center>Centered text</div>

<!-- Colors -->
<div text-red-500>Red text</div>
<div bg-blue-500>Blue background</div>
```

## Borders

```html
<div border border-gray-200>Border</div>
<div rounded-lg>Rounded corners</div>
<div shadow-lg>Shadow</div>
```

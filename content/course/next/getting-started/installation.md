# Installation

Install Next.js in your project.

## npm

```bash
npx create-next-app@latest
```

## yarn

```bash
yarn create next-app
```

## pnpm

```bash
pnpm create next-app
```

## bun

```bash
bunx create-next-app
```

## TypeScript

```bash
npx create-next-app@latest --typescript
```

## Tailwind CSS

```bash
npx create-next-app@latest --tailwind
```

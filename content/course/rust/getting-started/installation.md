# Installation

Install Rust using rustup and set up your development environment.

## Install with rustup

On macOS and Linux:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

On Windows, download and run rustup-init.exe from https://rustup.rs.

## Verify Installation

After installation, verify it's installed:

```bash
rustc --version
cargo --version
```

## What's Included

Rust includes:

- **rustc**: The Rust compiler
- **cargo**: Package manager and build system
- **rustup**: Toolchain installer
- **rustfmt**: Code formatter
- **clippy**: Linter

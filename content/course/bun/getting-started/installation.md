# Installation

Install Bun on your system using the official installer.

## Install on macOS/Linux

```bash
curl -fsSL https://bun.sh/install | bash
```

## Install on Windows

```powershell
irm bun.sh/install.ps1 | iex
```

## Verify Installation

```bash
bun --version
```

## Update Bun

```bash
bun upgrade
```

## Package Managers

You can also install Bun using package managers:

- **Homebrew**: `brew install oven-sh/bun/bun`
- **npm**: `npm install -g bun`
- **yarn**: `yarn global add bun`
- **pnpm**: `pnpm add -g bun`

# Installation

Install Nuxt in your project.

## npm

```bash
npx nuxi@latest init <project-name>
```

## yarn

```bash
yarn create nuxt-app <project-name>
```

## pnpm

```bash
pnpm dlx nuxi@latest init <project-name>
```

## bun

```bash
bunx nuxi@latest init <project-name>
```

## Existing Project

```bash
npm install nuxt
npm install -D @nuxtjs/tailwindcss
```

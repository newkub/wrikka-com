# Components

## DocSidebar

Component สำหรับแสดง navigation ซ้ายของหน้า docs โดยอัตโนมัติสร้างจากโครงสร้างโฟลเดอร์ `content/docs/`

## DocTOC

Component สำหรับแสดง Table of Contents ทางขวา โดยดึงจาก headings ใน markdown

## DocNav

Component สำหรับแสดงปุ่ม Previous/Next navigation ด้านล่าง

## DocContent

Component สำหรับ render markdown content พร้อม syntax highlighting
